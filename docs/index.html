Introduction to Migrate

What is Migrate?

Provides a framework for migrating complex data from a source (xml, json, sql) to a Drupal site.

Feeds can be used for simple migrations.

History of Migrate

Drupal 7

  • Drush support
  • Mature (built-in support for csv, sql, xml, etc)
  • Drupal-to-Drupal migration
  • Migrate Extras
  • Wordpress, TYPO3, etc

Drupal 8

  • Provides upgrade path from Drupal 6/7 to 8

Status of Migrate

  • Still considered experimental
  • No support for basic sources
  • No Drush support
  • Various process plugins

Contributed Modules for Migrate

  • Migrate Tools
  • Migrate Plus
  • Migrate Source CSV
